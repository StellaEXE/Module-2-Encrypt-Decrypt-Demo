# Encryption Demo

This script demonstrates basic symmetric and asymmetric encryption techniques in Python.

## Functionality
1.  **Symmetric Encryption (Fernet)**: Uses the same secret key to encrypt and decrypt the message. It is fast and efficient for small/large data.
2.  **Asymmetric Encryption (RSA)**: Uses a public key to encrypt and a corresponding private key to decrypt. Anyone can have the public key, but only the owner of the private key can decrypt.

## How to Run
1.  Install the required library: `pip install cryptography`
2.  Run the script: `python encryption_demo.py`

## Deliverables
- `encryption_demo.py`: Python code.
- `README.md`: This file.
- Console Output showing keys, inputs, and outputs.
